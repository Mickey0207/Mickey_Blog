## 一、標籤

### 1. html 網頁開始和結束

```html
<html></html>
```
   
### 2. head 標頭

它是元素與數據內容的容器，主要是寫給機器運作與搜尋使用的

```html
<head></head>
```
   
### 3. title 網頁視窗標題

網頁視窗標題開始和結束

```html
<title></title>
```
   
### 4. body 網頁內容

是網頁所有顯示內容的容器，是網頁真正要跑給使用者看的內容。

```html
<html>

<head>
    <title>標題放這裡</title>
</head>

<body>
    網頁內容
</body>
```
