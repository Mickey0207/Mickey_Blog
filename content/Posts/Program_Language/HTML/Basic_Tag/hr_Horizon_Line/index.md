## 一、標籤

```html
<hr>
```

----
## 二、屬性

### 1. 直接用顏色名稱換顏色

```html
<hr color = 'blue'>
```

### 2. 用十六進制換顏色

```html
<hr color = '#844200'>
```

### 3. 水平線高度

```html
<hr size = '50'>
```

### 4. 寬度設為500畫素

```html
<hr  width = "500">
```

### 5. 佔螢幕的50%

```html
<hr width = "50%">
```

### 6. 至左對齊

```html
<hr align="left">
```
