## 一、標籤

```html
<font>輸入文字</font>
```

---
## 二、屬性 

### 1. 字型

```html
<font face = "標楷體">AI整合應用奪先機。</font>
```

### 2. 字體大小

```html
<font size = "5">程式、影音、平面、插畫、遊戲、室內設計，</font>
```

### 3. 字體顏色

```html
<font size = "5">程式、影音、平面、插畫、遊戲、室內設計，</font>
```

### 4. 總體
```html
<font color = "blue" face = "標楷體" size = "5">資深業師乾貨不藏私，打造全領域 AI通才。</font>
```
