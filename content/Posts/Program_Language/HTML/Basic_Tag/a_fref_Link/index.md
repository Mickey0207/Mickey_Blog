## 一、標籤

```html
<a></a> 超連結顯示文字
```

## 二、屬性

### 1. 網址放在 `herf` 裡面

```html
<a href="https://www.ifreesite.com/color/#google_vignette">精選六大領域課程</a>
```

### 2. target 開新視窗

開啟超連結的時候開新視窗

```html
<a href="https://www.ifreesite.com/color/#google_vignette" target ="new">精選六大領域課程</a>
```
