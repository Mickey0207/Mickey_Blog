## 一、標籤

```html
<body>輸入網頁內容</body>
```

## 二、屬性

### 1. 控制背景顏色

```html
<body bgcolor = 'blue'>
```
   
### 2. 控制文字顏色

```html
<body text = "blue">
```  

### 3. 控制連結顏色

```html
<body link = "blue">
```   

### 4. 控制已閱讀過的顏色

```html
<body vlink = "blue">
```

### 5. 控制正在連結的顏色

```html
<body alink = "blue">
```   
   
### 6. 插入背景圖

```html
<body background="images/bg1.jpg" text="red" vlink="blue" alink="blue">
```
