## 一、說明

---
## 二、img 插入圖片

```html
<img>
```

---
## 三、img 屬性

### 1. src 路徑

可以使用 網址 或是 本機圖片

```html
<img src="images/bg1.jpg">
```

### 2. width 寬

```html
<img src="images/bg1.jpg" width="50%">
```

### 3. height 高

```html
<img src="images/bg1.jpg" height="80%">
```

### 4. align 對齊

```html
<img src="images/bg1.jpg" align=" right">
```
